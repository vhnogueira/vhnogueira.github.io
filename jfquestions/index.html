LET’S PLAY!!

Instructions:

Allow for one person to share their screen with the GAME LINK open.

Each person will answer the question that pops up on the screen (in as much detail as they are comfortable with).

Once you are done press shuffle and, simply move on to the next person to answer the next question.

If you are not comfortable answering a question OR you have already answered it, simply shuffle to the next question until you find a question you would like to answer.

Keep going until you run out of time.
